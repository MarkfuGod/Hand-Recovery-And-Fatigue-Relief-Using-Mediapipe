## Requirements:
_Make sure your `python` version >= `3.7.0`_
```shell

```
## How to run the game
### ChallengeMode:
```shell
cd ChallengeMode/
python main.py
```
### NormalMode:
```shell
cd NormalMode/
python main.py
```
### RecoveryMode:
```shell
cd RecoveryMode/
python main.py
```
### LoginPage:
***Make sure you have `npm` dependency***
```shell
cd vue-test1/
npm install
npm run dev
```
